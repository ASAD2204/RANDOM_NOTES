Revision Chunk B (07-12)

Hashing, trees, graphs, searching, sorting, recursion/backtracking.

📚 10-Minute Recall

  • Hash map average O(1), worst O(n).
  • Trees: inorder of BST gives sorted order.
  • Graphs: BFS/DFS are O(V+E).
  • Searching: linear vs binary (sorted prerequisite).
  • Sorting: know stable vs unstable and O(n log n) defaults.
  • Backtracking: choose → explore → undo.

⚠ Tricky Cases

  • Visited set forgotten in graph traversal.
  • Binary search boundary bugs.
  • JS numeric sort comparator omission.
  • Backtracking state not reverted.

🌟 Interview Tip

For traversal problems, say representation choice first (adjacency list / tree node model) before coding.

▶ Theory Deep Dive

Data structure and algorithm choices should be justified by complexity, memory behavior, and workload characteristics.
