Bit Manipulation

Fast bitwise tricks used in coding interviews.

📚 Concept

Use bitwise operators &, |, ^, ~, <<, >> for compact and efficient logic.

💡 Analogy

Binary control panel: each switch bit turns feature on/off instantly.

📄 Step-by-step tricks

Check odd: n & 1
Power of 2: n>0 and (n & (n-1))==0
Clear lowest set bit: n & (n-1)
Get lowest set bit: n & -n
XOR pair cancel: a^a=0

▶ Implementations

// C++ single number
int x=0; for(int v:arr) x^=v;

# Python bit count
bits = n.bit_count()

// JS power of two
const isPow2=n=>n>0 && (n&(n-1))===0;

⏱ Complexity

Bit operations are O(1) for fixed-width integers.

✅ / ❌

  • ✅ Very fast, elegant for certain patterns.
  • ❌ Can hurt readability if overused.

⚠ Common Mistakes

  • Confusing XOR (^) with exponentiation.
  • Signed shift behavior misunderstandings.
  • Ignoring integer width language differences.

🌟 Interview Tips

  • Must-know problems: single number, subset generation via bitmask, count set bits.

▶ Theory Deep Dive

Data structure and algorithm choices should be justified by complexity, memory behavior, and workload characteristics.
