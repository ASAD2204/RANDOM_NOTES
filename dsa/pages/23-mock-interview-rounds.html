Mock Interview Rounds

Concept + coding + optimization drills for timed interview simulation.

▶ Round A (45 min)

  • Easy warmup: array/string (10 min)
  • Medium pattern: sliding window or graph traversal (20 min)
  • Follow-up optimization + complexity discussion (15 min)

▶ Round B (60 min)

  • Design + implement custom data structure (LRU/min stack/trie)
  • Handle edge cases + explain trade-offs
  • Add one extension requirement

🌟 Self-Score Rubric

5/5 pattern identification + correct code + clear complexity
4/5 correct code, weak explanation
3/5 brute-force only
2/5 syntax/logic unstable
1/5 incomplete approach

▶ Theory Deep Dive

Data structure and algorithm choices should be justified by complexity, memory behavior, and workload characteristics.
