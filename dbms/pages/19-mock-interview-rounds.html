Mock Interview Rounds (DBMS + SQL)

Timed concept + query writing rounds for placement simulation.

▶ Round A (40 min)

  • 10 min theory: keys, normalization, ACID.
  • 20 min query writing: joins + group by + subquery.
  • 10 min output prediction and optimization follow-up.

▶ Round B (50 min)

  • Case study schema design.
  • Write 3 production-style queries.
  • Add index strategy and explain expected performance impact.

🌟 Self-score rubric

5/5 correct logic + clean SQL + strong optimization discussion
4/5 correct results, weaker performance explanation
3/5 mostly correct but misses edge/null handling
2/5 syntax and grouping mistakes
1/5 incomplete solution
