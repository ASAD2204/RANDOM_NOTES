Practice SQL Problems

Beginner to advanced SQL exercises for interviews and revision.

🟢 Beginner

  1. List all employees with salary > 60000.
  2. Find employees in Engineering department.
  3. Count employees per department.
  4. Show top 3 salaries.
  5. Find employees with NULL department.

🟡 Intermediate

  1. Department-wise highest salary employee.
  2. Employees earning above department average.
  3. Consecutive duplicate events in logs.
  4. Customers with no orders.
  5. Monthly revenue report using GROUP BY date truncation.

🔴 Advanced

  1. Nth highest salary per department.
  2. Running totals with window functions.
  3. Gap and island problems on date sequences.
  4. Recursive hierarchy query (org chart).
  5. Top K products per category with ties.

🧭 Practice strategy

Daily (45-60 min):
- 2 beginner refreshers
- 2 intermediate pattern questions
- 1 advanced or window-function problem

After solving:
- compare with alternate solution
- optimize readability + performance

🌟 Interview Tips

  • Always validate with sample output mentally before final query.
  • Use aliases and formatting to improve readability in interviews.
