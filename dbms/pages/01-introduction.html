Introduction to DBMS

What DBMS is, why it is used, DBMS types, and architecture basics.

📚 What is DBMS?

A Database Management System is software that stores, manages, secures, and retrieves data efficiently while supporting concurrency and integrity.

  • Examples: MySQL, PostgreSQL, Oracle, SQL Server, SQLite.

✅ Advantages over file systems

  • Reduced redundancy and inconsistency.
  • Multi-user concurrency control.
  • Data integrity via constraints.
  • Security via authentication and authorization.
  • Backup and recovery features.

🧩 Types of DBMS

  • Hierarchical DBMS
  • Network DBMS
  • Relational DBMS (RDBMS)
  • Object-oriented DBMS
  • NoSQL systems (document, key-value, graph, columnar)

🏗 Database architecture (3-level)

External level: user views
Conceptual level: logical schema
Internal level: physical storage details

This separation gives data independence.

🌟 Interview Tips

  • When asked DBMS vs file system, focus on ACID + concurrency + constraints.
  • Know logical vs physical data independence definitions.
