SQL Overview

SQL categories: DDL, DML, DQL, DCL, TCL with practical command examples.

📚 What is SQL?

Structured Query Language is the standard language used to define, manipulate, control, and query relational data.

🗂 SQL categories + commands

DDL: CREATE, ALTER, DROP, TRUNCATE
DML: INSERT, UPDATE, DELETE
DQL: SELECT
DCL: GRANT, REVOKE
TCL: COMMIT, ROLLBACK, SAVEPOINT

▶ Example commands

-- DDL
CREATE TABLE t(id INT PRIMARY KEY, name VARCHAR(30));
ALTER TABLE t ADD email VARCHAR(100);

-- DML
INSERT INTO t VALUES (1, 'A');
UPDATE t SET name='B' WHERE id=1;
DELETE FROM t WHERE id=1;

-- DQL
SELECT * FROM t;

-- TCL
START TRANSACTION;
SAVEPOINT s1;
ROLLBACK TO s1;
COMMIT;

🌟 Interview Tips

  • TRUNCATE removes all rows fast; usually cannot be rolled back in some DBs depending on engine/settings.
  • DELETE can use WHERE and logs row-level changes.
