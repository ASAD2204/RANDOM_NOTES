Common SQL Interview Questions

Frequently asked query patterns with practical SQL solutions.

▶ Sample table

employees(emp_id, emp_name, dept_id, salary)
(1,Asha,10,60000)
(2,Ravi,20,75000)
(3,Meera,10,80000)
(4,Karan,NULL,50000)
(5,Isha,20,75000)

▶ Second highest salary

SELECT MAX(salary) AS second_highest
FROM employees
WHERE salary < (SELECT MAX(salary) FROM employees);

Output: 75000

▶ Nth highest salary (N=3)

SELECT DISTINCT salary
FROM employees
ORDER BY salary DESC
LIMIT 1 OFFSET 2;

Output: 60000

▶ Find duplicates (by emp_name)

SELECT emp_name, COUNT(*) AS cnt
FROM employees
GROUP BY emp_name
HAVING COUNT(*) > 1;

Output: (empty for current sample)
-- In general returns duplicated names.

▶ Employees without departments

SELECT emp_id, emp_name
FROM employees
WHERE dept_id IS NULL;

Output:
4 Karan

🌟 Interview Tips

  • For ranking questions, mention alternatives: subquery, window functions, CTE.
  • Explain NULL handling explicitly.
