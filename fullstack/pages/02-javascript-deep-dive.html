JavaScript Deep Dive

Advanced concepts required for modern frontend and backend JS.

▶ Closures and lexical scope

function counter() {
  let c = 0;
  return () => ++c;
}
const inc = counter();
console.log(inc(), inc()); // 1 2

▶ Async model: callbacks, promises, async/await

const fetchUser = () => new Promise(r => setTimeout(() => r({id:1}), 200));
async function main(){
  const user = await fetchUser();
  console.log(user.id);
}
main();

▶ Event loop intuition

Call Stack + Web APIs + Task Queue + Microtask Queue
Microtasks (Promise jobs) run before next macrotask.

▶ Prototypal inheritance + ES6+

class Animal { speak(){ return "..."; } }
class Dog extends Animal { speak(){ return "woof"; } }
const d = new Dog();
console.log(d.speak());

ES6+ essentials: let/const, arrow functions, destructuring, spread/rest, modules, template literals.

▶ Theory Deep Dive

JavaScript behavior is governed by lexical scope and event-loop scheduling. Correct async code depends on understanding microtasks, macrotasks, and closure lifetime.
