Express.js Backend Development

Building REST APIs with middleware-driven architecture.

▶ Routing and middleware

import express from "express";
const app = express();
app.use(express.json());

app.get("/api/health", (req, res) => res.json({ok:true}));
app.post("/api/users", (req, res) => res.status(201).json(req.body));

▶ Request/response and error handling

app.use((err, req, res, next) => {
  res.status(err.status || 500).json({message: err.message || "Server error"});
});

▶ Auth and authorization

function auth(req, res, next){
  const token = req.headers.authorization?.split(" ")[1];
  if(!token) return res.status(401).json({message:"Unauthorized"});
  next();
}
app.get("/api/private", auth, (req,res)=>res.json({secret:true}));

▶ Theory Deep Dive

Express architecture is middleware composition. Reliability comes from clear routing, centralized errors, input validation, and observability-first API design.
