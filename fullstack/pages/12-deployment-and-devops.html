Deployment and DevOps

Shipping and operating full stack apps reliably.

▶ Environment variables

Keep secrets and environment-specific settings outside code (.env in dev, secret managers in prod).

▶ Docker + CI/CD

Build -> Test -> Security checks -> Deploy
(automated via GitHub Actions/GitLab CI/Jenkins)
FROM node:20-alpine
WORKDIR /app
COPY . .
RUN npm ci
CMD ["npm", "start"]

▶ Cloud hosting and scaling

  • Deploy frontend on CDN/static hosts.
  • Deploy APIs on containers/serverless/VM.
  • Scale using load balancers, autoscaling, cache layers.

▶ Theory Deep Dive

Deployment is an operations discipline: repeatable builds, immutable artifacts, staged rollout, and monitoring determine production reliability.
