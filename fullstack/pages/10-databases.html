Databases in Full Stack Applications

Integrating relational and document databases effectively.

▶ MySQL and PostgreSQL

Relational databases with ACID guarantees. PostgreSQL offers advanced SQL/extensibility; MySQL is widely used and performant for many workloads.

▶ MongoDB

Document database using flexible JSON-like documents, useful for schema flexibility and rapid iteration.

▶ ORM concepts

  • Object mapping between code models and DB tables/docs.
  • Migration tools manage schema changes.
  • Examples: Prisma/Sequelize/TypeORM (Node), SQLAlchemy (Python).

▶ Example query flow

React form submit -> POST /api/orders
API validation -> ORM create(order)
DB commit -> JSON response to frontend

▶ Theory Deep Dive

Database selection is a trade-off across consistency, flexibility, query patterns, and operational complexity. Schema and indexing strategy heavily affect system performance.
