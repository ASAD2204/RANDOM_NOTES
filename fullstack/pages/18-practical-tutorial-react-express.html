Practical Tutorial 1: Task App (React + Express)

Build a mini production-style task manager with CRUD APIs.

▶ Project architecture

frontend (React) -> /api/tasks (Express) -> PostgreSQL/MongoDB

▶ Step-by-step

  1. Initialize frontend and backend projects.
  2. Create task API routes: GET/POST/PUT/DELETE.
  3. Connect React UI with fetch/axios.
  4. Add form validation + loading/error states.
  5. Add auth middleware and protected routes.

▶ Express route example

router.post("/tasks", async (req, res) => {
  const task = await Task.create(req.body);
  res.status(201).json(task);
});

▶ React integration example

const createTask = async (payload) => {
  const res = await axios.post("/api/tasks", payload);
  setTasks(prev => [res.data, ...prev]);
};

▶ Theory Deep Dive

This project demonstrates end-to-end contract alignment between UI state transitions, API behavior, and persistence semantics.
