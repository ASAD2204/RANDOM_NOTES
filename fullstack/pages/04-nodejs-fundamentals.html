Node.js Fundamentals

JavaScript runtime for backend services and tooling.

▶ Runtime and internals

  • Built on V8 engine for JS execution.
  • libuv handles event loop, async I/O, and thread pool.
  • Single-threaded JS with non-blocking I/O for concurrency.

▶ Event-driven architecture

const EventEmitter = require("events");
const bus = new EventEmitter();
bus.on("order.created", (id) => console.log("process", id));
bus.emit("order.created", 42);

▶ Modules + npm ecosystem

// ESM
import express from "express";
// package management
npm install express dotenv

Node supports CommonJS and ESM module patterns.

▶ Theory Deep Dive

Node.js concurrency comes from non-blocking I/O and event-driven execution. Throughput gains require avoiding event-loop blocking and isolating CPU-heavy work.
