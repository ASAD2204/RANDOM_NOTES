Combinational Circuits

Output depends only on current input (no memory).

▶ Half adder and full adder

Half Adder:
Sum = A ⊕ B
Carry = A·B

Full Adder:
Sum = A ⊕ B ⊕ Cin
Carry = AB + Cin(A ⊕ B)

▶ Multiplexer (MUX)

A MUX selects one of many inputs using select lines (e.g., 4:1 MUX uses 2 select bits).

S1 S0 = 00 -> I0
S1 S0 = 01 -> I1
S1 S0 = 10 -> I2
S1 S0 = 11 -> I3

▶ Decoder and encoder

  • Decoder: n input lines to 2^n output lines (e.g., 2-to-4).
  • Encoder: reverse mapping, active input encoded to binary.
