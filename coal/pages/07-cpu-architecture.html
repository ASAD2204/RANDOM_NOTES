CPU Architecture

CPU components and how they collaborate per instruction.

▶ Core components

  • ALU: arithmetic and logical operations.
  • Control Unit (CU): decodes instruction and generates control signals.
  • Registers: very fast small storage (PC, IR, MAR, MDR, GPRs, flags).
  • Buses: data bus, address bus, control bus.

▶ Block view

          +----------------------+
Memory <-->| MAR MDR        CU   |
           |      Registers      |
           |          ALU        |
           +----------------------+

🌟 Architecture vs organization

ISA defines instructions visible to programmer; organization defines implementation (pipeline depth, cache policy, bus width).

▶ Theory Deep Dive

Computer organization topics are best understood as abstraction layers from digital logic to CPU, memory, and I/O performance trade-offs.
