Cache Memory

Small fast memory between CPU and RAM to reduce access latency.

▶ Mapping techniques

  • Direct-mapped: each memory block maps to exactly one cache line (simple, conflict misses).
  • Fully associative: block can go anywhere (best flexibility, expensive search).
  • Set-associative: compromise; block maps to one set, any line in that set.

▶ Performance terms

Hit ratio = hits / total accesses
Miss ratio = 1 - hit ratio
AMAT = Hit time + Miss rate × Miss penalty

🌟 Mini computation

If hit time=1 ns, miss rate=5%, miss penalty=50 ns, then AMAT = 1 + 0.05×50 = 3.5 ns.
