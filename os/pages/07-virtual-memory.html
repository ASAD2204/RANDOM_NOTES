Virtual Memory

Demand paging and page replacement algorithms: FIFO, LRU, Optimal.

📚 Concept

Virtual memory gives each process an illusion of large contiguous memory using disk + RAM.

Demand paging flow

1) CPU references page
2) If page not in RAM -> page fault
3) OS loads page from disk into free frame (or replace victim)
4) Update page table and resume process

▶ Page replacement algorithms

  • FIFO: evict oldest loaded page.
  • LRU: evict least recently used page.
  • Optimal: evict page used farthest in future (theoretical best).

Example reference string

Ref: 7 0 1 2 0 3 0 4
Frames: 3
Compare page faults under FIFO/LRU/Optimal for interview practice.

🌟 Interview Tips

  • Belady's anomaly can occur in FIFO.
  • LRU approximations used in practice due to overhead.

▶ Theory Deep Dive

Virtual memory decouples address space from physical RAM. Paging enables large-process abstraction but introduces page-fault latency and replacement trade-offs.
