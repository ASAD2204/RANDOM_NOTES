OS Security

Protection mechanisms and access control fundamentals.

📚 Security goals

  • Confidentiality
  • Integrity
  • Availability

Protection mechanisms

User authentication
Authorization (ACLs/capabilities)
Process isolation
Memory protection
File permissions
Auditing and logging

Access control models

  • Discretionary Access Control (DAC)
  • Mandatory Access Control (MAC)
  • Role-Based Access Control (RBAC)

🌟 Interview Tips

  • Explain principle of least privilege.
  • Kernel/user mode separation is core to OS protection.

▶ Theory Deep Dive

OS security is a layered defense model involving authentication, authorization, isolation, least privilege, and auditing. Kernel attack surface reduction is central.
