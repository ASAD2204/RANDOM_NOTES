Constructors and Destructors

Default, parameterized, copy constructor, and destructor behavior across languages.

▶ C++ Constructor Spectrum

class Box {
public:
  int w;
  Box() : w(0) {}                 // default
  Box(int x) : w(x) {}            // parameterized
  Box(const Box& b) : w(b.w) {}   // copy constructor
  ~Box() {}                       // destructor
};

📚 Cross-language notes

  • C++: deterministic destruction when object leaves scope.
  • Python: __init__ initializes; __del__ is non-deterministic (GC-driven).
  • JavaScript: constructor via class; no explicit destructors (GC handles memory).

⚠ Interview traps

  • Shallow copy of owning pointers in C++ causes double-free bugs.
  • Assuming Python __del__ always runs immediately.
  • Ignoring move semantics in modern C++ when discussing copies.
