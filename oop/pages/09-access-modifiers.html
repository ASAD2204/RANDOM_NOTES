Access Modifiers

Public, private, protected and their role in encapsulation.

📄 Visibility Table

Modifier     Class   Subclass   Outside
public       ✔       ✔          ✔
protected    ✔       ✔          ✖
private      ✔       ✖          ✖

▶ C++ Example

class User {
private: string passwordHash;
protected: int loyaltyPoints;
public: string name;
};

🌟 Interview Insight

Encapsulation is enforced through private fields + controlled public methods. “Everything public” is usually a design smell.

▶ Theory Deep Dive

Object-oriented design should be explained through abstraction boundaries, coupling/cohesion, and maintainability trade-offs.
